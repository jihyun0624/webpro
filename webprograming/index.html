Python

(반복문과 조건문 예제)

목차

반복문

  • for
  • while

조건문

  • if
  • elif
  • else

반복문 for 

for i in range(1, 10):
    print(i)

반복문 while 

while i > 0:
    print(i)
    i -= 1

조건문 if 

if i > 10:
    print('i는 10보다 작습니다.')

조건문 elif

if i > 10:
    print('i는 10보다 작습니다.')

elif i < 20:
    print('i는 20보다 큽니다.')

조건문 else 

if i > 10:
    print('i는 10보다 작습니다.')
            
elif i < 20:
    print('i는 20보다 큽니다.')

else:
    print('i는 10보다 크고 20보다 작습니다.')
